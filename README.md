## 🧠 فكرة المشروع

تهدف منصة عِتاق إلى:
- تعزيز الوعي بالتراث العمراني المحلي
- إتاحة التفاعل مع المباني التراثية رقمياً
- تمكين الوصول للتراث من أي مكان في العالم
